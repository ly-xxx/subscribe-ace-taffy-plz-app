Release APK

现在就能打出 release 包，没证书也能先做体验分发。

Quick Start 
powershell -ExecutionPolicy Bypass -File .\scripts\build-release-apk.ps1

默认支持 debug keystore fallback，适合真机体验包；准备好正式证书后可以无缝切换。

Output

What is included

  • docs/index.html 静态 landing page

Deploy Flow

发布步骤

  1. 把仓库推到 GitHub。
  2. 在仓库里启用 GitHub Pages / Actions。
  3. 工作流会自动把 docs/ 发布成站点。

默认工作流监听 master 分支推送，也支持手动触发。
